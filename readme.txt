Pokémon API Server
Este proyecto es un servidor básico de Node.js utilizando Express que permite obtener una lista de los primeros 100 Pokémon desde la
PokeAPI
.

Descripción
El servidor proporciona un endpoint que devuelve un JSON con los nombres y URLs de los primeros 100 Pokémon. Utiliza Axios para realizar la solicitud a la API pública de Pokémon y manejar la respuesta adecuadamente.

Requisitos
Node.js (v12 o superior)
npm (o yarn)
